# CLI Proxy API 管理中心（改进版）

基于原项目进行体验优化，当前版本主要更新如下：

## 更新内容

1. API 密钥支持别名  
可在新增/编辑 API 密钥时填写别名，方便区分不同用途的密钥。

2. 未保存配置离开提醒  
检测到未保存更改时，离开页面会弹窗确认，避免误操作导致配置丢失。

3. 使用统计支持显示 API 别名  
在使用统计相关视图中可显示 API 别名，定位请求来源更直观。

## 配置方法

1. 打开管理面板并进入 `远程管理`。  
2. 在 `面板仓库` 中填写：`https://github.com/jie8190/Cli-Proxy-API-Management-Center`。  
3. 保存配置并按需重启服务使配置生效。
